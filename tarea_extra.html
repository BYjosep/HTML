Tarea extra: etiquetas vistas

Las etiquets vistas son las siguientes:
p: Esta etiqueta te permite hacer paragrafos. Todo este texto esta siendo escrito en un unico paragrafo.

br: Esta etiqueta te permite hacer un salto de linea. Como es el caso de este paragrafo.

a: Esta etiqueta sirve para poner enlaces como el siguiente: Este enlace te redirigira a Aules de FP. En esta etiqueta tambien se pueden poner correos electronicos, telefonos y enlaces que se habran en una pestaña nueva.

li: Esta etiqueta junto al las etiquetas ol y ul te permiten crear listas hordenadas en el caso del uso de ol y desordenadas en el caso de ul. Aqui como ejemplo el codigo de la tarea listas:

Lista ordenada

  1. Despertarse
  2. Comprar cerveza
  3. Comprar cerveza Hasta que no quede el supermecado

Lista desordenada

  • Patatas
  • Cerezas
  • Agua


img: Esta etiqueta te permite puner fotos ya se que las tengas almacenadas en tu servidor web o este en internet.


div: Esta etiqueta sirve para poder hacer que no hayan etiquetas colisionandose entr si. Como el caso de la imagen que ha sido subida en la explicazion de la etiqueta img.

table: Esta etiquetasirve para crear tablas junto a las etiquetas thead y tbody y dentro de ellas td y tr. Tambien dentro de la etiqueta table se a usado el modificador border con valor = 1. Dentro de estas dos ultimas etiquetas se les puede añadir dos modificadores: colspan y rowspan. Aqui una pequeña tabla.
cabecera de tabla
Celda 1 Celda 2
Celda 3 Celda 4


strong: Esta etiqueta sirve para resaltar o poner en negrita cosas importantes como es el caso de el nombre de las etiquetas en todo este texto.

center: Esta etiqueta es usada para centrar un texto como en el caso de la tarea tablas para poner el texto centrado en las celdas Horario y patio.

Las etiquetas que se generan al escribir html:5 no van a ser explicadas ya que estas son esenciales para que html se pueda representar de forma correcta , exceptuando la etiqueta meta que esta sirve entre otros para poner el autor de la pagina y para indicar al navegador si quieres que la pagina este indexada.
